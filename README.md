# Cómo poner en marcha la app.

- Para hacerlo en modo debbug primero hacer desde la carpeta "Gestionart" docker compose up --build -d

- Corremos únicamente el contenedor de sql.

- En VS Code abromos la carpeta "API" que está en : "Gestionart/gestionart_backend_ruben_y_jessica/api"

- Desde el main "ApiAplication" ejecutamos en modo debug

- Con postman lanzamos las peticiones con esta base "http://localhost:8080"
